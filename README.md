# Netflix Clone

A full-stack Netflix-inspired streaming platform featuring user authentication, protected routes, and responsive user interfaces.

## Features

- User Registration & Login
- JWT Authentication
- Protected Routes
- Session Management
- Responsive Netflix-style UI
- MongoDB Integration

## Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js, MongoDB, JWT Authentication
